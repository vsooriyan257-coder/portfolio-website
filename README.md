# Portfolio Website

This is my full-stack portfolio website developed as part of my academic project.

## Technologies Used
- HTML
- CSS
- Node.js
- MongoDB

## Features
- Single page portfolio website
- Contact form
- Backend server using Node.js
- Data stored in MongoDB database

## How to Run the Project
1. Install Node.js
2. Install MongoDB and start it
3. Open terminal in project folder
4. Run: npm install
5. Run: node server.js
6. Open browser and go to: http://localhost:3000
